## Sign up

可以在该页面注册新的账号。

![signup](../../resource/advanced-guides/web-console/login/signup.png)

**Account**: 注册用户名，长度为3-10。

**Email**: 邮箱地址。

**Password**: 至少 8-16 个字符，其中至少 1 个小写字母和 1 个数字。

**ConfirmPassword**: 确认密码。
